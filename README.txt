================================================================================
          基于网络的彩票购买抽奖系统 —— 使用说明
================================================================================

【环境要求】
  - JDK 8 或更高版本
  - 命令行终端 (CMD / PowerShell / Git Bash)

【快速启动】

  步骤1：编译
    打开终端，进入项目目录，执行:
    javac -encoding UTF-8 -d bin src/common/*.java src/server/*.java src/client/*.java

    或者 Windows 下直接双击运行: compile.bat

  步骤2：启动服务器
    java -cp bin server.LotteryServer

    服务器启动后会显示:
    ========================================
      彩票服务器已启动，端口: 8888
      最大并发线程数: 50
    ========================================
      管理员命令: draw=摇奖 stats=统计 exit=退出

  步骤3：启动客户端 (可打开多个终端窗口，启动多个客户端)
    java -cp bin client.LotteryClient

【操作流程】

  客户端操作:
    1. 先注册账号(1→2)，再登录(1→1)
    2. 登录后先充值(2)，再购买彩票(1)
    3. 选号格式: 01 05 12 18 25 33  (6个红球, 空格分隔)
    4. 蓝球: 08  (1个数字)
    5. 随时查看购彩记录(3)和中奖通知(4)

  服务器管理员操作:
    - 输入 draw → 摇奖开奖
    - 输入 stats → 查看当期统计
    - 输入 exit → 关闭服务器

【测试多客户端并发】
  打开3个终端，都启动客户端，即可模拟3个用户同时在线

【常见问题】
  Q: 提示"无法连接到服务器"
  A: 确保先启动了 LotteryServer

  Q: 端口冲突
  A: 修改 LotteryConstants.java 中的 SERVER_PORT

  Q: 中文乱码
  A: 编译时加 -encoding UTF-8 参数
